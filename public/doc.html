Popis rolí

V systému jsou role úředník, policie, soud, administrátor.

Administrátor může dělat vše.

Uředníci evidují řidiče, vozidla, řidičská oprávnění. Úředníci dále ukládají do systému bonusové body řidičům (ty se odečíatají), které mohou získat např. za obsolvování školení z bezpečné jízdy.

Policie eviduje přestupky řidičů a krádeže vozidel. Policie také do systému vkládá informaci, zda bylo kradené vozidlo nelezeno a vráceno majiteli.

Soud může měnit nebo rušit přestupky řidičů.

Instalace

Softwarové požadavky

  • Ruby 1.9
  • Rails 3
  • Bundler
  • Rake
  • postgresql

Instalace

  • rozbalit aplikaci unzip xdujic01.zip / unzip xcupak04.zip
  • nainstalování potřebných rubygems bundle install
  • nakonfigurovat databázi v souboru config/database.yml
  • vytvoření databáze rake db:create
  • vytvoření tabulek databáze rake db:migrate
  • naplnění databáze daty rake db:seed
  • spuštění aplikace rails server
